Deklarace pole

Deklaruje pole

let pole = [1, 2, 3];
let pole = new Array();

Naplnění pole

Cyklus for

let pole = [];
for (let i = 0; i < pole.length; i++){
pole[i] = "hodnota"; }

length

Vrací počet prvků v poli

let pole = [1, 2, 3];
console.log(pole.length); // Vypíše 3

push()

Přidá jednu nebo více položek na konec pole a vrátí novou délku pole.

let pole = [1, 2, 3];
let novaDelka = pole.push(4, 5);
console.log(novaDelka); // Vypíše 5 console.log(pole); // Vypíše [1, 2, 3, 4, 5]

shift()

Odstraní první prvek z pole a vrátí ho.

let pole = [1, 2, 3];
let prvniPrvek = pole.shift();
console.log(prvniPrvek); // Vypíše 1 console.log(pole); // Vypíše [2, 3]

unshift()

Přidá jednu nebo více položek na začátek pole a vrátí novou délku pole.

let pole = [1, 2, 3];
let novaDelka = pole.unshift(-1, 0);
console.log(novaDelka); // Vypíše 5 console.log(pole); // Vypíše [-1, 0, 1, 2, 3]

join()

Spojí všechny prvky pole do řetězce pomocí zadaného oddělovače.

let pole = [1, 2, 3];
let spojenyRetezec = pole.join('-');
console.log(spojenyRetezec); // Vypíše "1-2-3"

toString()

Vrátí řetězcovou reprezentaci pole.

let pole = [1, 2, 3];
let retezec = pole.toString();
console.log(retezec); // Vypíše "1,2,3"

concat()

Spojí dvě nebo více polí a vytvoří nové pole.

let pole1 = [1, 2];
let pole2 = [3, 4];
let novePole = pole1.concat(pole2);
console.log(novePole); // Vypíše [1, 2, 3, 4]

indexOf()

Vrátí první index, na kterém je nalezen zadaný prvek v poli.

let pole = [1, 2, 3, 2, 1];
let index = pole.indexOf(2);
console.log(index); // Vypíše 1

lastIndexOf()

Vrátí poslední index, na kterém je nalezen zadaný prvek v poli.

let pole = [1, 2, 3, 2, 1];
let index = pole.lastIndexOf(2);
console.log(index); // Vypíše 3

sort()

Seřadí prvky pole podle abecedního nebo numerického pořadí.

let pole = [3, 1, 2];
pole.sort();
console.log(pole); // Vypíše [1, 2, 3]

reverse()

Obrátí pořadí prvků v poli.

let pole = [1, 2, 3];
pole.reverse();
console.log(pole); // Vypíše [3, 2, 1]
